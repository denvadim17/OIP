<p><b>Самши́т вечнозелёный</b> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">Búxus sempervírens</span>),&#160;— <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%B5%D1%81%D0%BD%D1%8B%D0%B5_%D1%80%D0%B0%D1%81%D1%82%D0%B5%D0%BD%D0%B8%D1%8F" title="Древесные растения">древесное растение</a>, <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B8%D0%B4" title="Биологический вид">вид</a> рода <a href="/wiki/%D0%A1%D0%B0%D0%BC%D1%88%D0%B8%D1%82" title="Самшит">Самшит</a> семейства <a href="/wiki/%D0%A1%D0%B0%D0%BC%D1%88%D0%B8%D1%82%D0%BE%D0%B2%D1%8B%D0%B5" title="Самшитовые">Самшитовые</a>. <i>пальмовым деревом</i><sup id="cite&#95;ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup>.

<p>Самшит вечнозелёный&#160;— <a href="/wiki/%D0%94%D0%B5%D1%80%D0%B5%D0%B2%D0%BE" title="Дерево">дерево</a> высотой до 15&#160;м, иногда <a href="/wiki/%D0%9A%D1%83%D1%81%D1%82%D0%B0%D1%80%D0%BD%D0%B8%D0%BA" title="Кустарник">кустарник</a>.
</p><p><a href="/wiki/%D0%9F%D0%BE%D0%B1%D0%B5%D0%B3_(%D0%B1%D0%BE%D1%82%D0%B0%D0%BD%D0%B8%D0%BA%D0%B0)" title="Побег (ботаника)">Побеги</a> прямые, торчащие, четырёхгранные, зелёные, густо покрыты листьями.
</p><p><a href="/wiki/%D0%9B%D0%B8%D1%81%D1%82" title="Лист">Листья</a> на коротких <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D1%88%D0%BE%D0%BA" title="Черешок">черешках</a>, почти сидячие, сильно различающиеся по форме, голые, с блестящей тёмно-зелёной верхней стороной и светло-зелёной, даже желтоватой нижней.
</p><p><a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BE%D0%BA" title="Цветок">Цветки</a> мелкие, зеленоватые, обычно однополые, <a href="/wiki/%D0%A2%D1%8B%D1%87%D0%B8%D0%BD%D0%BA%D0%B0" title="Тычинка">тычиночные</a>&#160;— в компактных <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%BE%D0%B2%D0%BA%D0%B0_(%D1%81%D0%BE%D1%86%D0%B2%D0%B5%D1%82%D0%B8%D0%B5)" class="mw-redirect" title="Головка (соцветие)">головчатых соцветиях</a>, <a href="/wiki/%D0%9F%D0%B5%D1%81%D1%82%D0%B8%D0%BA" title="Пестик">пестичные</a>&#160;— одиночные<sup id="cite&#95;ref-flower&#95;3-0" class="reference"><a href="#cite_note-flower-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup>.
</p><p><a href="/wiki/%D0%9F%D0%BB%D0%BE%D0%B4" title="Плод">Плод</a>&#160;— мелкая чёрно-коричневая трёхрожковая шаровидная <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%B1%D0%BE%D1%87%D0%BA%D0%B0" title="Коробочка">коробочка</a> с выростами, раскрывающаяся по мере созревания <a href="/wiki/%D0%A1%D0%B5%D0%BC%D1%8F" title="Семя">семян</a> створками.
</p>

<p>Произрастает в <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Западная Европа">Западной</a> и <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Южная Европа">Южной Европе</a>, северо-западной <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">Африке</a> и <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%B4%D0%BD%D1%8F%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Передняя Азия">Передней Азии</a>. Его <a href="/wiki/%D0%90%D1%80%D0%B5%D0%B0%D0%BB" title="Ареал">ареал</a> простирается от южной <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D1%8F" title="Англия">Англии</a> до северного <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BE%D0%BA%D0%BA%D0%BE" title="Марокко">Марокко</a>, а на востоке достигает <a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турции</a> и <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7" title="Западный Кавказ">Западного Кавказа</a>.
</p>
