<p><b>Лен Ве́стерботтен</b> (<a href="/wiki/%D0%A8%D0%B2%D0%B5%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Шведский язык">швед.</a> <span lang="sv" style="font-style:italic;">Västerbottens län</span>) — <a href="/wiki/%D0%9B%D0%B5%D0%BD_(%D0%B0%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0)" title="Лен (административная единица)">лен</a> на севере <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швеции</a>. Граничит с ленами <a href="/wiki/%D0%95%D0%BC%D1%82%D0%BB%D0%B0%D0%BD%D0%B4_(%D0%BB%D0%B5%D0%BD)" title="Емтланд (лен)">Емтланд</a>, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%BD%D0%BE%D1%80%D1%80%D0%BB%D0%B0%D0%BD%D0%B4" title="Вестерноррланд">Вестерноррланд</a> и <a href="/wiki/%D0%9D%D0%BE%D1%80%D1%80%D0%B1%D0%BE%D1%82%D1%82%D0%B5%D0%BD_(%D0%BB%D0%B5%D0%BD)" title="Норрботтен (лен)">Норрботтен</a>. Административный центр — город <a href="/wiki/%D0%A3%D0%BC%D0%B5%D0%BE" title="Умео">Умео</a>.
</p><p>Лен полностью включает в себя территорию <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D0%B8_%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D0%B8" title="Провинции Швеции">исторической провинции</a> <a href="/wiki/%D0%92%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%B1%D0%BE%D1%82%D1%82%D0%B5%D0%BD_(%D0%BF%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D1%8F)" title="Вестерботтен (провинция)">Вестерботтен</a>, южную часть <a href="/wiki/%D0%9B%D0%B0%D0%BF%D0%BF%D0%BB%D0%B0%D0%BD%D0%B4_(%D0%BF%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D1%8F)" title="Лаппланд (провинция)">Лаппланда</a> и небольшую часть на севере <a href="/wiki/%D0%9E%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4" title="Онгерманланд">Онгерманланда</a>.
</p>

<div class="mw-heading mw-heading2"><h2 id="География"><span id=".D0.93.D0.B5.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span>География</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%92%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%B1%D0%BE%D1%82%D1%82%D0%B5%D0%BD_(%D0%BB%D0%B5%D0%BD)&amp;veaction=edit&amp;section=1" title="Редактировать раздел «География»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%B1%D0%BE%D1%82%D1%82%D0%B5%D0%BD_(%D0%BB%D0%B5%D0%BD)&amp;action=edit&amp;section=1" title="Редактировать код раздела «География»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>На северо-западе лена расположены <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D0%B0%D0%B2%D1%81%D0%BA%D0%B8%D0%B5_%D0%B3%D0%BE%D1%80%D1%8B" title="Скандинавские горы">Скандинавские горы</a>, с которых через весь лен на юго-запад в <a href="/wiki/%D0%91%D0%BE%D1%82%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BB%D0%B8%D0%B2" title="Ботнический залив">Ботнический залив</a> течёт множество рек. Крупнейшие из них — <a href="/wiki/%D0%9E%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%8D%D0%BB%D1%8C%D0%B2%D0%B5%D0%BD" title="Онгерманэльвен">Онгерманэльвен</a>, <a href="/wiki/%D0%A8%D0%B5%D0%BB%D0%BB%D0%B5%D1%84%D1%82%D0%B5%D1%8D%D0%BB%D1%8C%D0%B2%D0%B5%D0%BD" title="Шеллефтеэльвен">Шеллефтеэльвен</a> и <a href="/wiki/%D0%A3%D0%BC%D0%B5%D1%8D%D0%BB%D1%8C%D0%B2%D0%B5%D0%BD" title="Умеэльвен">Умеэльвен</a>, а также приток последней <a href="/w/index.php?title=%D0%92%D0%B8%D0%BD%D0%B4%D0%B5%D0%BB%D1%8D%D0%BB%D1%8C%D0%B2%D0%B5%D0%BD&amp;action=edit&amp;redlink=1" class="new" title="Винделэльвен (страница отсутствует)">Винделэльвен</a>. Также есть множество озер, крупнейшее из которых — <a href="/w/index.php?title=%D0%A1%D1%82%D1%83%D1%80%D1%83%D0%BC%D0%B0%D0%BD_(%D0%BE%D0%B7%D0%B5%D1%80%D0%BE)&amp;action=edit&amp;redlink=1" class="new" title="Стуруман (озеро) (страница отсутствует)">Стуруман</a>.

<p>Область знаменита высокими горами, преобладающими на северо-западе, где расположены популярные среди шведских отпускников горные курорты <a href="/w/index.php?title=%D0%A5%D0%B5%D0%BC%D0%B0%D0%B2%D0%B0%D0%BD&amp;action=edit&amp;redlink=1" class="new" title="Хемаван (страница отсутствует)">Хемаван</a> и <a href="/w/index.php?title=%D0%A2%D0%B5%D1%80%D0%BD%D0%B0%D0%B1%D1%8E&amp;action=edit&amp;redlink=1" class="new" title="Тернабю (страница отсутствует)">Тернабю</a> — родные места многих известных спортсменов, в том числе горнолыжников <a href="/wiki/%D0%90%D0%BD%D1%8F_%D0%9F%D0%B5%D1%80%D1%81%D0%BE%D0%BD" class="mw-redirect" title="Аня Персон">Ани Персон</a> и <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D0%BC%D0%B0%D1%80_%D0%A1%D1%82%D0%B5%D0%BD%D0%BC%D0%B0%D1%80%D0%BA" class="mw-redirect" title="Ингемар Стенмарк">Ингемара Стенмарка</a>, прославивших Швецию большим количеством международных титулов и призов.
</p><p>В городе Шеллефтео играет одна из сильнейших команд <a href="/wiki/%D0%A8%D0%B2%D0%B5%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D1%8D%D0%BB%D0%B8%D1%82%D0%BD%D0%B0%D1%8F_%D1%81%D0%B5%D1%80%D0%B8%D1%8F" class="mw-redirect" title="Шведская элитная серия">шведской хоккейной Элитсерии</a> — <a href="/wiki/%D0%A8%D0%B5%D0%BB%D0%BB%D0%B5%D1%84%D1%82%D0%B5%D0%BE_%D0%90%D0%98%D0%9A" class="mw-redirect" title="Шеллефтео АИК">Шеллефтео АИК</a>.
</p>
