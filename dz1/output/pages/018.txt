						<div id="siteSub" class="noprint">Материал из Википедии — свободной энциклопедии</div>

					<div id="contentSub"><div id="mw-content-subtitle"></div></div>
					
					
					<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="ru" dir="ltr"><div role="note" class="hatnote navigation-not-searchable noprint dabhide">В Википедии есть статьи о других людях с именем <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_III" class="mw-disambig" title="Иоанн III">Иоанн III</a>.</div>
<p><b>Иоанн III</b>&#160;— восьмой <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Архиепископ">архиепископ</a> <a href="/wiki/%D0%A1%D0%BE%D0%BB%D1%8C%D1%82%D0%B0%D0%BD%D0%B8%D0%B5" title="Сольтание">Сольтание</a> (с <a href="/wiki/1398_%D0%B3%D0%BE%D0%B4" title="1398 год">1398 года</a>), <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA%D0%B0%D0%BD%D1%86%D1%8B" title="Доминиканцы">доминиканец</a>; дипломат, путешественник и писатель; знаток нескольких восточных языков. О его национальности, месте и времени рождения нет точных данных. Часто отождествляется с <b>Иоанном де Галонифонтибусом</b> (de Galonifontibus, то есть выходец из <a href="https://fr.wikipedia.org/wiki/Gaillefontaine" class="extiw" title="fr:Gaillefontaine">Gaillefontaine</a> в <a href="/wiki/%D0%92%D0%B5%D1%80%D1%85%D0%BD%D1%8F%D1%8F_%D0%9D%D0%BE%D1%80%D0%BC%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Верхняя Нормандия">Верхней Нормандии</a>), епископом <a href="/wiki/%D0%9D%D0%B0%D1%85%D0%B8%D1%87%D0%B5%D0%B2%D0%B0%D0%BD" class="mw-redirect" title="Нахичеван">Нахичевана</a> (с <a href="/wiki/1377" class="mw-redirect" title="1377">1377</a>); иногда&#160;— с английским <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%B0%D0%BD%D1%86%D1%8B" title="Францисканцы">францисканцем</a> Джоном Гринлоу (John Greenlaw), епископом <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%B4%D0%B0%D0%B9%D1%8F" class="mw-redirect" title="Солдайя">Солдайи</a>. Некоторые историки сомневаются в корректности таких отождествлений
<sup id="cite&#95;ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
<sup id="cite&#95;ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> .
</p><p>26 августа 1398&#160;г. папа <a href="/wiki/%D0%91%D0%BE%D0%BD%D0%B8%D1%84%D0%B0%D1%86%D0%B8%D0%B9_IX" title="Бонифаций IX">Бонифаций IX</a> утвердил Иоанна, находившегося в <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Риме</a>, на <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F_%D0%A1%D0%BE%D0%BB%D1%8C%D1%82%D0%B0%D0%BD%D0%B8%D0%B5" title="Архиепархия Сольтание">архиепископскую кафедру Сольтание</a> в <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BD" title="Иран">Иране</a>, завоёванном к тому времени <a href="/wiki/%D0%A2%D0%B0%D0%BC%D0%B5%D1%80%D0%BB%D0%B0%D0%BD" title="Тамерлан">Тимуром</a>. Иоанн отправился на восток, но после победы Тимура над <a href="/wiki/%D0%91%D0%B0%D1%8F%D0%B7%D0%B8%D0%B4_I" title="Баязид I">Баязидом</a> <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BE%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B1%D0%B8%D1%82%D0%B2%D0%B0" title="Ангорская битва">при Ангоре</a> (июль <a href="/wiki/1402_%D0%B3%D0%BE%D0%B4" title="1402 год">1402&#160;г.</a>) был послан победителем в Европу с известием об этом событии. Посетив <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%86%D0%B8%D1%8F" title="Венеция">Венецию</a> и <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%83%D1%8F" title="Генуя">Геную</a>, Иоанн в мае <a href="/wiki/1403_%D0%B3%D0%BE%D0%B4" title="1403 год">1403 года</a> оказался в <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a>, а затем&#160;— в <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a>. Он передал королям <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_VI_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Карл VI (король Франции)">Карлу VI Французскому</a> и <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_IV_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B8)" title="Генрих IV (король Англии)">Генриху IV Английскому</a> письма Тимура и его сына <a href="/wiki/%D0%9C%D0%B8%D1%80%D0%B0%D0%BD-%D1%88%D0%B0%D1%85" title="Миран-шах">Миран-шаха</a>, в которых предлагалось заключить договора о взаимовыгодной торговле.
</p><p>В июне <a href="/wiki/1407_%D0%B3%D0%BE%D0%B4" title="1407 год">1407 года</a> Иоанн находился в Венеции, летом <a href="/wiki/1408" class="mw-redirect" title="1408">1408</a>&#160;— в <a href="/wiki/%D0%9F%D0%B8%D0%B7%D0%B0" title="Пиза">Пизе</a>, затем с делегацией участников <a href="/wiki/%D0%9F%D0%B8%D0%B7%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Пизанский собор">Пизанского собора</a> посетил <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D0%B5%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE" class="mw-redirect" title="Венгерское королевство">Венгрию</a>, где были проведены переговоры с королём <a href="/wiki/%D0%A1%D0%B8%D0%B3%D0%B8%D0%B7%D0%BC%D1%83%D0%BD%D0%B4_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Сигизмунд (император Священной Римской империи)">Сигизмундом</a> по поводу <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%80%D0%B0%D1%81%D0%BA%D0%BE%D0%BB" title="Великий западный раскол">церковного раскола</a>. В последующие годы он, вероятно, проявлял активность в установлении союзнических отношений между Сигизмундом и <a href="/wiki/%D0%A2%D0%B8%D0%BC%D1%83%D1%80%D0%B8%D0%B4%D1%8B" title="Тимуриды">тимуридами</a> Миран-шахом и <a href="/wiki/%D0%A8%D0%B0%D1%85%D1%80%D1%83%D1%85" title="Шахрух">Шахрухом</a>. В <a href="/wiki/1410_%D0%B3%D0%BE%D0%B4" title="1410 год">1410&#160;г.</a>, будучи в Риме, Иоанн был назначен главой архиепископства <a href="/wiki/%D0%A5%D0%B0%D0%BD%D0%B1%D0%B0%D0%BB%D1%8B%D0%BA" title="Ханбалык">Ханбалыка</a> (<a href="/wiki/%D0%9F%D0%B5%D0%BA%D0%B8%D0%BD" title="Пекин">Пекина</a>) в сане <a href="/wiki/%D0%9C%D0%B8%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82" title="Митрополит">митрополита</a>. Около <a href="/wiki/1423_%D0%B3%D0%BE%D0%B4" title="1423 год">1423 года</a> выехал из Ирана в <a href="/wiki/%D0%9A%D1%80%D1%8B%D0%BC" title="Крым">Крым</a>, где и провёл последние годы жизни, курируя также крымскую епархию.
</p><p>Иоанн&#160;— автор <i><a rel="nofollow" class="external text" href="http://www.vostlit.info/Texts/rus13/Galonifontibus/frametext.htm">Книги познания мира</a></i> (<i>Libellus de notitia orbis</i>), написанной в <a href="/wiki/1404_%D0%B3%D0%BE%D0%B4" title="1404 год">1404 году</a> на испорченной <a href="/wiki/%D0%9B%D0%B0%D1%82%D1%8B%D0%BD%D1%8C" class="mw-redirect" title="Латынь">латыни</a> и являющейся важным источником по истории и этнографии <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7" title="Кавказ">Кавказа</a>, <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Малая Азия">Малой Азии</a>, Ирана и других регионов. В нём также содержатся данные о деятельности в этих странах доминиканцев и францисканцев. В своих описаниях Иоанн основывался как на собственных путевых впечатлениях, так и на информации, полученной от других путешественников, в основном, купцов. Кроме того, он пользовался трудами средневековых авторов, таких как <a href="/wiki/%D0%9F%D1%91%D1%82%D1%80_%D0%95%D0%B4%D0%BE%D0%BA" class="mw-redirect" title="Пётр Едок">Пётр Едок</a>, <a href="/wiki/%D0%93%D0%BE%D0%BD%D0%BE%D1%80%D0%B8%D0%B9_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%BE%D0%B4%D1%83%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Гонорий Августодунский">Гонорий Августодунский</a> и <a href="/wiki/%D0%98%D1%81%D0%B8%D0%B4%D0%BE%D1%80_%D0%A1%D0%B5%D0%B2%D0%B8%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9" title="Исидор Севильский">Исидор Севильский</a>. До настоящего времени сохранилось три рукописи сочинения архиепископа Иоанна.
</p>
